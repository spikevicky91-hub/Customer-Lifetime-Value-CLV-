#Customer Lifetime Value (CLV) Analytics Project

Built a customer lifetime value analytics platform using a modern data stack, where I calculated CLV, ARPU, and performed segmentation across channels and regions using dbt models on top of a 5000-record dataset.

#Overview
Production-grade CLV analytics system:
1. 5000+ transaction records
2. Customer revenue tracking
3. Lifetime value calculation
4. Segmentation by channel & region

#KPIs
1. Customer Lifetime Value (CLV)
2. Average Revenue per User (ARPU)
3. Customer Segmentation
